## 数据结构

- 连续结构
- 跳转结构
- 连续+跳转

### 概念

- 数据结构是存储、组织数据的方式

- 精心选择的数据结构可以带来更高的运行或者存储效率

- 数据结构是很多算法得以进行的载体

最基本的数据结构

- 数组：便于寻址
- 链表：便于更改
